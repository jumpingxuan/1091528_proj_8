# 1091528_proj_8
IoT_樹梅派搭配Button操控LED實作

實作過程:

1.接好電路

2.按照題目指示，

  Pattern 1: 兩個LED燈亮；Pattern 2: 兩個LED燈不亮；Pattern 3: 兩個LED輪流閃爍
  
  利用1個Button切換三種模式，每按一次Button就換下一個Pattern。
  
3.設計python程式

特別注意:

在pattern 3的部分time.sleep(0.2)是因為做完上一個pattern後，會有雜訊產生，所以需要過一段時間後，再開始執行。
